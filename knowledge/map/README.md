# Knowledge Map

## Purpose

`map/` holds the broad knowledge-cluster gateways for the public Knowledge atlas.

The current map also hosts the instantiated 70-node seed program under those gateways.

## How to use it

- Move from broad cluster to narrower domains as notes are added.
- Use cluster gateways for orientation, not as substitutes for detailed study notes.
- Follow typed scaffold links from gateways into member nodes.
- Follow typed links when you want lateral movement rather than deeper nesting.

## Navigation rule

Use cluster gateways for:

- orientation
- member-node access
- typed scaffold links into the node graph
- typed movement to adjacent clusters

Use node notes for:

- focused study
- canonical anchor gathering
- lateral movement through typed relations
