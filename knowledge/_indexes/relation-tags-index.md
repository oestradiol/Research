## Usage rule

Prefer typed links when the relation matters. Do not use relation tags just to decorate obvious folder proximity.

Use `scaffolds` from cluster gateways down to member nodes.

Use `emerges-from` from a node back up to its cluster gateway, or between clusters/domains when one area is being described as developmentally arising out of another.

Use `requires` and `supports` to connect study routes to each other.

Every cluster gateway and every instantiated seed node should use these tags to stay visible in graph view as one connected structure rather than a set of isolated folders.
