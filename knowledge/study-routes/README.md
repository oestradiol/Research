# Study Routes

## Purpose

Study routes provide optional guided traversals through the public Knowledge graph.

## How to use them

- Use a route when you want a bounded reading path through several clusters.
- Treat routes as guides, not as replacements for the notes themselves.
- Allow routes to combine SUF with other frameworks where that improves the study path.
- Follow route-to-route dependencies when you want a more cumulative progression.
- Use Backlinks and Outgoing Links to inspect how routes and nodes support each other.
- If you want a course-design or self-study framing around the routes, start with [Studying and Teaching with SUF](../studying-and-teaching-with-suf.md).
