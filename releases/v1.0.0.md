# Research `v1.0.0` Release Notes

Date: `2026-04-02`

Repository: [oestradiol/Research](https://github.com/oestradiol/Research)

Tag target: `v1.0.0`

## Summary

`v1.0.0` is the first bounded public release of the full `Research/` repository. It publishes:

- [Structured Unity Framework](https://github.com/oestradiol/Research/tree/main/structured-unity-framework) as the academic core
- [Knowledge](https://github.com/oestradiol/Research/tree/main/knowledge) as the sibling atlas founded on SUF
- [Research Tools](https://github.com/oestradiol/Research/tree/main/tools) as the read-only validation and reporting layer

## Release highlights

- SUF now exposes a four-layer public framework stack, a bounded New Zealand demonstrated route, a bounded Taiwan starter comparator, and a locked public payoff sentence.
- Knowledge now ships as a stable 9-cluster / 70-node public graph with typed relations and study routes.
- Research Tools now validate repo-wide links, version surfaces, route consistency, source-registry structure, and release readiness.
