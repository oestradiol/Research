# Applications Atlas

This layer is the public bridge between the domain-agnostic framework and concrete research use.

It has two distinct subtrees:

- **demonstrated routes**: bounded cases or route families where SUF already has a concrete public attachment
- **research map**: a targeted atlas of plausible next domains, methods clusters, and comparative directions

The breadth of the atlas is intentional, but it does **not** imply that SUF is already validated across every mapped area.

## 1. How to navigate this layer

- Start with [demonstrated-routes/README.md](demonstrated-routes/README.md) if you want the active case package.
- Start with [research-map/README.md](research-map/README.md) if you want the wider research atlas.
- Use [_indexes/applications-index.md](_indexes/applications-index.md) for a flattened navigation map.
- Use [_indexes/relation-tags-index.md](_indexes/relation-tags-index.md) to understand the typed relation vocabulary.

## 2. Status boundaries
