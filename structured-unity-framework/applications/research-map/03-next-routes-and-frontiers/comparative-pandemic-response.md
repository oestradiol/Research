# Comparative Pandemic Response

## Role in the applications atlas

This node is the first bounded comparative extension of the New Zealand route.

## Why SUF applies here

Pandemic response cases provide visible perturbation windows, formal decision structures, and publicly documented sequencing that make cross-case comparison tractable if the route stays bounded.

## Main objects of study

- national response architectures
- escalation and de-escalation timing
- border, tracing, and communication bundles
- institutional variation across comparable crisis episodes

## Typical units and unitization problems

Units are often not directly portable across countries. The main issue is building a comparator design that preserves functional equivalence without pretending that unitary and federal or different preparedness contexts are the same.

## Typical scales

- micro: matched decision or implementation events
- meso: functional clusters across cases
- macro: comparative whole-route signatures

## Likely I / C / L / U relevance

This node uses all four variables comparatively. `U` should never be treated as a universal country ranking; it is a bounded cross-window and cross-case interpretive signature.

## One concrete SUF research question

How do New Zealand, Taiwan, and a bounded Australian comparator differ in dependency density, sequencing stability, and lag when early border and public-health decisions are aligned against comparable acute-response windows?

## One bounded future route

A three-step expansion: first New Zealand plus Taiwan, then a Commonwealth-focused Australian contrast, then a carefully delimited broader comparator set only if unit-matching remains defensible.
