# Research Map

This subtree is a **targeted public atlas**, not a claim to have mapped the whole sciences or validated SUF across every domain.

## How to read it

- Cluster files explain why a family of domains matters.
- Node files give a bounded research sketch for a specific domain or subdomain.
- Typed relations make lateral movement explicit rather than collapsing everything into a generic “related” link.
