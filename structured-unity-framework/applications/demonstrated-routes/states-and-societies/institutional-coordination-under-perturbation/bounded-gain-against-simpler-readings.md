# Bounded Gain Against Simpler Readings

## 1. Purpose

This note states the **bounded empirical gain** of the current New Zealand route against two simpler readings only:

1. **executive-command-only**
2. **communication-as-downstream**

It does not function as a field-settling rival-framework chapter. It is a route-local clarification note attached to the current public evidence base.

## 2. Current evidence base used here

This comparison is limited to the currently public New Zealand package:

- the `38`-event public ledger
- the `29`-event main perturbation interval
- the first-pass `I`, `C`, and `L` companions
- the conjoint window comparison
- the `14`-check sensitivity and null note
- the current direct DPMC architecture, public-information, and late-transition review anchors

This note does **not** assume later Australia work, later closure work, or a mature cross-framework literature chapter.

## 3. Rival A: executive-command-only

### 3.1 What that simpler reading already explains

The executive-command-only reading can already explain:

- that central executive actors were highly salient
- that formal decisions at the strategic centre mattered
- that the response included visible command shifts and authority concentration
- that acute crisis response often looks more centralized than ordinary governance

That reading is therefore not wrong in full. It captures part of the route.

### 3.2 What SUF adds beyond it

The current SUF route adds three things that the executive-command-only reading flattens:

1. **cross-unit dependency structure**
   - the route is not only a chronology of centre-issued commands
   - it shows repeated routing across public-health, emergency-management, border, enforcement, public-service, and public-information surfaces

2. **public-information coordination as structurally central**
   - public-information coordination is not just a press wrapper around command
   - it remains coupled to escalation, implementation thickening, easing preparation, and de-escalation

3. **window-and-scale sensitivity**
   - the route distinguishes Comparator A, the acute perturbation interval, and Comparator B
   - this makes it possible to ask how organizational form changes rather than only who issued orders

### 3.3 What in the current evidence supports that added load

The current public package supports that added load because:

- all default `sigma1` units are active in the route
- cross-cluster routing is visible in the dependency summaries
- public-information coordination receives `33 / 38` coded events in the current public ledger
- the route does not collapse under the current robustness and null checks when architecture-heavy observations are perturbed
- the DPMC action-plan and public-information surfaces now directly support the architecture claim rather than leaving it only to retrospective reconstruction

### 3.4 What would weaken or erase that added value

The SUF gain over executive-command-only would weaken if later evidence showed that:

- most of the apparent cross-unit structure is curation noise rather than durable routing
- public-information coordination becomes peripheral once later additions are admitted
- perturbation checks show that the route can be explained equally well by a mostly central-command chronology
- comparator work shows the current gain is unique to one unusually curated case presentation

## 4. Rival B: communication-as-downstream

### 4.1 What that simpler reading already explains

The communication-as-downstream reading can already explain:

- that governments communicate decisions after decisions are made
- that messaging can help stabilize compliance and public understanding
- that visible information surfaces often follow backstage coordination

That reading is also not fully wrong. Public communication does often act downstream of prior decision processes.

### 4.2 What SUF adds beyond it

The current SUF route adds a narrower but important distinction:

1. **public-information coordination appears inside the coordination architecture**
   - it is not only downstream output
   - it shows up as part of the working response structure itself

2. **public-information coupling persists across phases**
   - escalation
   - implementation-side thickening
   - easing preparation
   - de-escalation

3. **coordination form matters, not only message presence**
   - the question is not just whether communication happened
   - it is whether communication was structurally coupled to routing, authority, and timing

### 4.3 What in the current evidence supports that added load

The current public package supports that added load because:

- public-information coordination is repeatedly active as a receiving or linked unit
- the route’s coherence note already reads public-information coupling as part of the coordination sequence rather than a decorative wrapper
- the DPMC public-information operating note now directly supports that role in the architecture layer
- the current Taiwan comparator does not reverse the pattern by making communication merely peripheral in the bounded comparator surface

### 4.4 What would weaken or erase that added value

The SUF gain over communication-as-downstream would weaken if later evidence showed that:

- the apparent centrality of public-information coordination depends on a narrow coding choice
- public-information links disappear under reasonable recoding or boundary perturbation
- stronger comparator work shows the same pattern is universal enough to add no explanatory distinction
- later source additions show that the real dependency structure was elsewhere and communication was mostly output-stage packaging

## 5. What this note does not claim

This note does not claim:

- that SUF defeats every nearby rival framework
- that the New Zealand route validates SUF across all institutional domains
- that these two simpler readings are the only possible comparators
- that bounded empirical gain here settles philosophy, ontology, or consciousness attribution

## 6. Working conclusion

At the current public baseline, SUF earns a **bounded** gain over these two simpler readings because it makes a specific structural claim visible:

> the New Zealand response should not be read only as centralized executive command, because public-information coordination is structurally central to the working coordination architecture.

That gain is real enough to state publicly, but narrow enough that later comparator work could still weaken, qualify, or partly erase it.
