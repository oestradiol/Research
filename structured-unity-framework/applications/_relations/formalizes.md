# formalizes

Use this relation when one node supplies the tools, model classes, or formal language needed to make another node operational.

Example: [network and dynamical systems](../research-map/00-foundations-and-methods/network-and-dynamical-systems.md) `formalizes` [infrastructure and logistics coordination](../research-map/03-next-routes-and-frontiers/infrastructure-and-logistics-coordination.md).
