# interprets

Use this relation when one node provides the meaning, evaluative frame, or explanatory context for another.

Example: [public information and communication](../research-map/01-states-public-health-and-governance/public-information-and-communication.md) `interprets` [public administration and executive coordination](../research-map/01-states-public-health-and-governance/public-administration-and-executive-coordination.md).
