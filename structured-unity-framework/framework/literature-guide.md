# Literature Guide

## 1. Purpose

This document is the public package's citation-routing layer. It says which source families support which moves, where those sources live in the public registry, and what they do not justify.

It is not a bibliography dump, a substitute for argument, or a prestige shield.

## 2. Control surfaces

Use the literature layer in this order:

1. `../references/citation-and-archival-policy.md` for citation shape
2. `../references/source-registry.md` for canonical source metadata and durable links
3. `../references/minimal-reading-list.md` for first orientation
4. `../references/standard-reading-list.md` for the active public anchor set
5. `../references/extended-reading-list.md` for targeted deepening only

## 3. Core routing rules

### Non-justification rule

No source family supports more than it actually supports.

- phenomenology supports awareness-first discipline, not the whole framework
- information theory and cybernetics support formal and organizational language, not lived meaning by themselves
- official case sources support boundary, timing, and architecture, not omniscient causal truth
- comparative literature supports matched comparison design, not country ranking by default

### False ancestry rule

Overlap is not lineage. Use bridge citations and contrasts when helpful, but do not claim direct ancestry where there is only partial resemblance.

### Public-legibility rule

Use as much literature as the file needs and no more. Public SUF should stay legible without turning every note into a literature review.

## 4. Core framework routing

### Structural Phenomenology

**Canonical sources** (genealogy and methodological precedent):

- Husserl, E. (1913). *Ideas Pertaining to a Pure Phenomenology and to a Phenomenological Philosophy, First Book* (§§24–29, the *epoché*). Trans. Kersten, 1982; trans. Dahlstrom, 2014.
- James, W. (1890). *The Principles of Psychology*, Chapter 9: "The Stream of Thought." Dover, 1950; Harvard UP, 1983 (critical edition).
- Merleau-Ponty, M. (1945). *Phenomenology of Perception*, Part 1: "Introduction." Trans. Landes, 2012.

**Contemporary registry family** (SUF direct sources):

- [src-gallagher-zahavi-phenomenological-mind-2012](../references/source-registry.md#src-gallagher-zahavi-phenomenological-mind-2012)
- [src-zahavi-subjectivity-selfhood-2005](../references/source-registry.md#src-zahavi-subjectivity-selfhood-2005)
- [src-varela-embodied-mind-1991](../references/source-registry.md#src-varela-embodied-mind-1991)
- [src-varela-neurophenomenology-1996](../references/source-registry.md#src-varela-neurophenomenology-1996) (methodological bridge to empirical research)

**Supports:**

- awareness-first discipline (as methodological choice, not novel metaphysics)
- first-person structure and embodiment
- constraints on what later operational layers may claim
- weak naturalization (phenomenology constrains models without being reduced to metrics)

**Does not justify:**

- a final ontology
- a direct metric for unity-like organization
- idealism as settled doctrine
- anti-scientific stance

## 7. AI-language routing

The public package treats "AI slop" as a writing-quality and evidence problem, not a detector problem.

Use these registry families when that issue is live:

- [src-acm-generative-ai-detection-2023](../references/source-registry.md#src-acm-generative-ai-detection-2023) for detector-caution policy
- [src-cheng-ai-detection-tools-2025](../references/source-registry.md#src-cheng-ai-detection-tools-2025) for empirical limits on detector and human identification
- [src-munoz-ortiz-llm-news-text-2024](../references/source-registry.md#src-munoz-ortiz-llm-news-text-2024) and [src-kobak-llm-biomedical-writing-2025](../references/source-registry.md#src-kobak-llm-biomedical-writing-2025) for measurable linguistic-pattern differences and homogenization tendencies
- [src-zerilli-transparency-trust-ai-2022](../references/source-registry.md#src-zerilli-transparency-trust-ai-2022) and [src-schilke-ai-disclosure-trust-2025](../references/source-registry.md#src-schilke-ai-disclosure-trust-2025) for transparency, disclosure, and trust effects
- [src-bender-koller-climbing-nlu-2020](../references/source-registry.md#src-bender-koller-climbing-nlu-2020) for meaning-versus-form caution

These sources support three public-writing rules:

- critique arguments, evidence, and inflation, not vibes
- avoid detector language as if it were dependable adjudication
- prefer concrete, bounded, source-bearing prose over generic polish

## 10. Citation-intensity modes

### Minimal

Use for entry points and overview docs. State roles and limits without overloading the reader.

### Standard

Use for research-ready public docs, the demonstrated route, and atlas nodes that need real starter anchors.

### Extended

Use only when a real objection, specialist audience, or frontier defense requires more depth.

## 11. What this file does not do

This file does not answer the question, `What is the total bibliography of the project?` It answers the narrower and more useful question, `Which sources go where, for what reason, and with what limits?`
