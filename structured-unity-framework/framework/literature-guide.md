# Literature Guide

## 1. Purpose

This document is the public package's citation-routing layer. It says which source families support which moves, where those sources live in the public registry, and what they do not justify.

It is not a bibliography dump, a substitute for argument, or a prestige shield.

## 2. Control surfaces

Use the literature layer in this order:

1. `../references/citation-and-archival-policy.md` for citation shape
2. `../references/source-registry.md` for canonical source metadata and durable links
3. `../references/minimal-reading-list.md` for first orientation
4. `../references/standard-reading-list.md` for the active public anchor set
5. `../references/extended-reading-list.md` for targeted deepening only

## 3. Core routing rules

### Non-justification rule

No source family supports more than it actually supports.

- phenomenology supports awareness-first discipline, not the whole framework
- information theory and cybernetics support formal and organizational language, not lived meaning by themselves
- official case sources support boundary, timing, and architecture, not omniscient causal truth
- comparative literature supports matched comparison design, not country ranking by default

### False ancestry rule

Overlap is not lineage. Use bridge citations and contrasts when helpful, but do not claim direct ancestry where there is only partial resemblance.

### Public-legibility rule

Use as much literature as the file needs and no more. Public SUF should stay legible without turning every note into a literature review.

- [src-sep-relational-quantum-mechanics-2025](../references/source-registry.md#src-sep-relational-quantum-mechanics-2025) and [src-sep-quantum-mechanics-2025](../references/source-registry.md#src-sep-quantum-mechanics-2025) for interpretation background and caution
- [src-ferrante-adversarial-consciousness-theories-2025](../references/source-registry.md#src-ferrante-adversarial-consciousness-theories-2025) for the current consciousness-testing baseline
- [src-garisto-quantum-echoes-willow-2025](../references/source-registry.md#src-garisto-quantum-echoes-willow-2025) only as an intuition aid for the echo / scrambling picture

These sources support:

- demystifying Willow-like hype into measurable many-body structure
- a bounded philosophical pressure toward relation / information / constraint-first readings
- a systems-information-relational consciousness line that keeps ontology separate from neural mechanism

These sources do not support:

- direct quantum-consciousness claims
- many-worlds proof claims
- a unique settled ontology
- substrate hardening beyond current public IAF limits

This branch should stay later-horizon and explicitly post-proof until the demonstrated-route and comparative work are stronger.

## 10. Citation-intensity modes

### Minimal

Use for entry points and overview docs. State roles and limits without overloading the reader.

### Standard

Use for research-ready public docs, the demonstrated route, and atlas nodes that need real starter anchors.

### Extended

Use only when a real objection, specialist audience, or frontier defense requires more depth.

## 11. What this file does not do

This file does not answer the question, `What is the total bibliography of the project?` It answers the narrower and more useful question, `Which sources go where, for what reason, and with what limits?`
