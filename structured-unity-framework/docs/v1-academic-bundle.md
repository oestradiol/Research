# v1 Academic Bundle

## Purpose

This note is the public academic front door for the current `v1` line of Structured Unity Framework. The hosted bounded tag is now `v1.1.0`; current `main` is aligned with that New Zealand monograph-baseline release point inside the same line.

## Abstract

Structured Unity Framework (SUF) is a layered, documentation-first framework for studying unity-like organization in complex systems without collapsing phenomenological, modeling, dynamic, and operational claims into one register. The current `v1` line presents **three substantive framework layers plus one bridge/control layer**, an explicit methods posture, a monograph-baseline New Zealand route core, and a read-only reproducibility layer. The main empirical anchor is a New Zealand pandemic-coordination route with a `38`-event public ledger, a `29`-event main perturbation interval, direct DPMC architecture and review anchors, route-local `I`, `C`, and `L` readouts, a dependency graph, a conjoint window comparison, and a `14`-check sensitivity/null note. A bounded Taiwan comparator now adds a `15`-event archive-clean tranche under the same source-admission rule. The strongest current payoff is not universal proof but a concrete case-level distinction: the New Zealand response should not be read only as centralized executive command, because public-information coordination is structurally central to the working coordination architecture. The package is positioned as methods/framework work first and philosophy/theory second. It offers a disciplined public architecture, a chapter-ready New Zealand core, a bounded comparator, and read-only validation tooling, while keeping stronger cross-case closure explicitly deferred rather than unresolved.

## Layer posture

- **Structural Phenomenology** = epistemic ground
- **Informational Awareness Framework** = bounded modeling and bridge vocabulary, not a finished technical theory
- **Unity Dynamics Framework** = most technical-facing layer, but still underdetermined, route-local, and not a closed predictive theory
- **Framework Interface** = bridge/control layer that keeps movement across the stack explicit and bounded

## Locked payoff sentence

**SUF shows that the New Zealand response should not be read only as centralized executive command, because public-information coordination is structurally central to the working coordination architecture.**

## Contribution claims

1. SUF supplies a disciplined layered framework that keeps phenomenological constraint, bounded modeling vocabulary, dynamic readout, and claim-type boundary control explicit instead of blending them into one vague theory voice.
2. SUF shows one bounded case-level analytical gain by making public-information coordination visible as a routed structural layer rather than treating it as downstream messaging around executive command.
3. SUF provides a reproducible documentation-first workflow in which source admission, event coding, window comparison, robustness checks, and read-only validation reports can all be inspected publicly.

## Reader expectation

Read this package as a bounded framework-and-methods release with one demonstrated route and one bounded comparator.

Do not read it as:

- a universal proof of SUF
- a closed empirical monograph
- a full rival-theory adjudication across the entire consciousness, systems, or institutional-methods literature
- a package that already offers objectively settled cross-domain measurement or high-fit predictive closure
