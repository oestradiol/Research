# Reviewer Objections and Current Answers

## Purpose

This file collects the strongest foreseeable academic objections to the current public package, the current repo-grounded answer to each, and the unresolved remainder that still survives after the answer.

It is not a promise that every objection is fully closed. It is a pressure-index for honest review.

## How to read this file

For each objection, this file states:

- the objection in plain language
- the current answer the repo can support
- the current evidence status
- the unresolved remainder
- the primary repo surfaces where that issue is handled

## 1. "This is just idealism with extra steps."

### Current answer

The current public package does not present itself as finished metaphysics. Structural Phenomenology is treated as an epistemic ground and constraint layer, while Framework Interface explicitly blocks silent hardening from modeling language into final ontology.

### Current evidence status

Partially handled at the framework-architecture level.

### Unresolved remainder

The repo is stronger at marking the philosophical boundary than at proving why this phenomenological start should be preferred over all rival epistemic starts.

### Primary surfaces

- `framework/framework-interface.md`
- `framework/structural-phenomenology-downstream-role.md`
- `docs/claims-and-boundaries.md`
