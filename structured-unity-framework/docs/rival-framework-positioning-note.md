# Rival-Framework Positioning Note

## Purpose

This file is the stable surface for the later bounded argument about what SUF explains better, or differently, than nearby public-administration, crisis-governance, and executive-command readings.

It is not yet a finished comparative-theory chapter.

## Current status

Current `main` is strong enough to justify a bounded gain claim inside the New Zealand route and the early Taiwan comparison, but it is not yet strong enough to sustain a mature rival-framework positioning chapter.

This file therefore functions as a planning scaffold.

## Planned comparison set

The later positioning argument should default to these nearby frames:

- executive-command-only readings of pandemic response
- communication-as-downstream or messaging-only readings
- broad crisis-governance summaries that describe coordination without route-level dependency structure
- public-administration accounts that explain institutional layering without a conjoint `I` / `C` / `L` reading

If future work adds more rivals, they should be added explicitly rather than smuggled in through broad rhetoric.

## Core positioning questions

The later chapter should answer:

1. What does SUF make visible that executive-command-only readings flatten?
2. What does SUF show about public-information coordination that messaging-only readings underdescribe?
3. What does route-level dependency analysis add beyond broad crisis-governance narrative summary?
4. Where do rival frameworks still outperform SUF, or explain different things that SUF is not trying to explain?
5. Which parts of the SUF gain depend on the New Zealand case only, and which survive the later Taiwan and Australia comparison?

## Evidence required before promotion

This note should not become a live positioning chapter until all of these exist:

- a monograph-grade New Zealand chapter set
- a chapter-ready Taiwan route
- a bounded Australia comparator
- a bounded pandemic-governance closure note that already states the traveling versus non-traveling patterns
- explicit supporting references for the rival readings actually being compared

## Guardrails

The later positioning argument should:

- compare arguments, not vibes or broad labels
- distinguish description, explanation, and claim strength
- avoid treating rival incompleteness as total failure
- avoid treating SUF as a field-settling theory

It should not:

- use weak caricatures as rivals
- imply universal superiority for SUF
- turn bounded empirical gain into cross-domain closure

## Promotion rule

This file should remain a scaffold until the closure note, claims surfaces, and comparator chapters are all strong enough to support a bounded cross-framework argument without rhetorical inflation.
