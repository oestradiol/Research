## What the grounding does

It provides the current scientific and methodological rationale for:
- the layered framework posture
- the bounded contribution shape
- the demonstrated New Zealand route
- the current evidence and boundary logic
- the weakening, null, and comparator posture

## What it does not do

It does not turn the package into universal proof.
It does not erase the difference between current demonstrated routes and future work.
It does not make the ontology final.
It does not let validator strength stand in for scientific closure.
