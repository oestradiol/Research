# Release checklist v0.1

Before calling the current SUF package clean and governed:

1. from the repository root, run `python package_doctor.py`
2. from the repository root, run `python research_launch_gate.py`
3. ensure the package authoritative index matches live package entrypoints
4. ensure package integrity hashes are current
5. ensure current package claims remain bounded
6. ensure package status, execution-order, and pending-inventory surfaces still point to the correct canonical docs
