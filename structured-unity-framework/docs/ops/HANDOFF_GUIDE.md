# Handoff guide

If you are handing off the current SUF package state to a future operator:

1. start with `../../START_HERE.md`
2. read `../project-status.md`
3. read `../current-execution-order.md`
4. use `../../governance/AUTHORITATIVE_INDEX_v0_1.md` before trusting any file as current
5. use `../../framework/research-program.md` and `../../references/source-registry.md` for methods and sources
6. from the repository root, run `python package_doctor.py` before treating the package as clean current state

Do not silently merge current truth, supportive context, and historical trace into one undifferentiated surface.
